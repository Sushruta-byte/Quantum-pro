An advanced, browser-based Augmented Reality (AR) Heads-Up Display (HUD) and neural assistant built using state-of-the-art computer vision, 3D graphics, and speech synthesis pipelines. Inspired by Stark Industries design architecture, this system maps biometric configurations to render immersive overlays directly in your web browser.  ⚡ Key FeaturesReal-time Biometric Tracking: Leverages MediaPipe Face Mesh to run face landmarker tracking directly in-browser, capturing micro-expressions and facial depth.  Three.js 3D Engine Integration: Renders highly optimized, responsive wireframe geometric models that dynamically scale, rotate, and align with the user's head position in real time.  Voice Command Engine: Utilizes the Web Speech API (SpeechRecognition and SpeechSynthesis) to provide natural language interactions and auditory status updates.  Intelligent Eye-Closure Safety Alarm: Calculates eye aspect metrics to detect closed eyes; remaining closed for over 2.5 seconds triggers a high-priority UI strobe visual alert alongside an auditory alarm generated via the Web Audio API.  Context-Aware Operation Modes: Features three distinct interface modules (Standard, Stealth, and Quantum MK-X) that scale sensory tracking, overlay states, and microphone capture settings on the fly.  🛠️ Tech StackFrontend: HTML5, Tailwind CSS  Computer Vision: MediaPipe (Face Mesh, Camera Utilities)  Graphics Engine: Three.js (WebGL rendering, clipping planes, and geometric meshes)  Audio & Speech Ecosystem: Web Speech API (webkitSpeechRecognition), Web Audio API (AudioContext Synthesizer)  🚀 Getting StartedPrerequisitesSince this interface requires camera and audio processing permissions, it must be served over a secure domain (https://) or a local development environment (http://localhost).
